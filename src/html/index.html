  • ctrl + C — остановить сборку, выйти из режима наблюдения

Команды для вызова задач по отдельности:

  • gulp html — Обработать html файлы
  • gulp sсss — Обработать scss файлы
  • gulp css — Обработать css файлы (Можно подключить вместо scss)
  • gulp js — Обработать js файлы
  • gulp img — Ужать изображения, подготовить webp версию
  • gulp font — Перевести шрифты в форматы woff, woff2
